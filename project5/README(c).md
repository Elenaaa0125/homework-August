## 三、实验原理  

### 3.1 ECDSA正常签名流程


1. 计算消息哈希: e = H(m)
 
2. 生成临时密钥: R = k·G

3. 计算签名:

   r = R.x mod n
 
   s = k⁻¹(e + dr) mod n


### 3.2 中本聪签名伪造方法

1. 随机选择 u, v ∈ [1, n-1]

2. 计算伪造点: R = u·G + v·P

3. 构造伪造参数:
  
   r = R.x

   s = r·v⁻¹ mod n

   e = r·u·v⁻¹ mod n
 
4. 验证时满足:

   s⁻¹(e·G + r·P) = R
